## Description:

This game is my 5th project for the Udaicty Front-End Web Dev nanodgree. The goal is to create a classic arcage frogger game using the newly acquired skills in Object Oriented JS and HTML5 Canvas. Images and a game loop engine are provided. The rest has been built in app.js file by Szofi with â™¥!

## Instructions
### How to play

* Use the arrow keys to move your player across the board.
* Avoid collision with the bugs, or you will have to start over.
* Keep going up until you reach the water to win the game.
